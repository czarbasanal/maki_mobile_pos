Handoff 13 · Logs · migration

Activity Logs — on the new language

The admin-only audit trail migrated to the elevated system. Flat Material Container rows with hairline dividers become soft-shadow AppCard day groups; the leading emoji glyph becomes a semantic per-type Lucide icon in a tinted tile. The color discipline is formalized, not loosened — color is reserved for audit-meaningful events only: security (login · access · user & role changes · password) reads red, financial (sale · void · refund) reads green, everything else stays neutral — with full dark parity. Date grouping, the app-bar type filter, the active-filter chip, and the read-only trail are all preserved.

Lucide — semantic per-type icons replace Cupertino + emoji glyphs
AppCard — soft-shadow day groups replace flat Container rows
Color reserved — security · financial · neutral
Read-only · admin-only — immutable trail, viewUserLogs
