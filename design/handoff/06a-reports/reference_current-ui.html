Pre-redesign (mixed). The Reports family — Sales History list, Sales Report dashboard, Profit Report, and Top Selling — is partly on the redesigned language already: it uses theme-aware colorScheme, dark-hairline parity, outlined metric/rank cards, semantic accents (success-green profit, error-red voids) and the medal idiom for top-3 ranks. What's not migrated: icons are still Cupertino, container surfaces are Material Card (flat shadow-mat elevation) rather than the soft-shadow AppCard, and there are no Lucide glyphs. Currency renders grouped (₱1,234.00) from the app-wide formatter. This bundle migrates all four surfaces to Lucide + AppCard while preserving the existing color discipline + dark parity — same pass done for POS / Checkout / Sale-Detail / Inventory / Receiving. Mark up the redesign you want and hand back. (Icons here render in Lucide — the migration target — even though the live screens still ship Cupertino.)
