Settings — admin editors

Slice 10b — the four admin-only sub-screens: Manage Lists (one tile per category kind), the per-kind Category editor CRUD, its edit form dialog, Cost Code Settings, and the Mechanics editor. Outlined elevation-0 Material Card rows and the native Switch / AlertDialog converge on the soft-shadow `AppCard` language; Cupertino glyphs → Lucide. The neutral-by-default discipline holds — color is the slate/gold primary (FAB · Save · code outline) and success-green on the encoded test chips; inactive items stay struck-through + grey (never deleted, so admin can reactivate). Save & Reset on Cost Codes are password-gated and log activity. Full dark parity (gold leads).
