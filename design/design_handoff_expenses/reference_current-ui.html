MAKI POS — Current UI · 08 · Expenses

Pre-redesign (raw Material). The Expenses feature (dashboard + add/edit form + grouped history) is plain Material: a category AppDropdown filter, a three-up compact SummaryCard totals row (Today / This Week / This Month), and lists of Material Card + ListTile rows. Icons are Cupertino (doc_plaintext on every row, sun_max / calendar / chart_bar on the totals). There is no AppCard, no Lucide, and no category color semantics — every expense uses the same neutral document glyph. Rows are swipe-to-delete (admin) with a hard-coded red background; tap edits; long-press also deletes. The form is a flat SingleChildScrollView of TextFormFields + dropdowns; delete confirmations are AlertDialogs. This bundle migrates it onto the elevated theme — Lucide + soft-shadow AppCard rows with dark parity, neutral-by-default discipline. Currency renders grouped (₱1,234.00). Mark up the redesign you want and hand back. (Icons here render in Lucide — the migration target — even though the live screens still ship Cupertino.)
