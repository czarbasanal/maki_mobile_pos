Handoff 08 · Expenses · migration

Expenses — on the new language

The expense tracker migrated to the elevated system — the dashboard (category filter · three-up totals · Recent list), its empty state, the add / edit form, the delete confirm + swipe-to-delete reveal, and the grouped history. Material Card + ListTile rows become soft-shadow AppCard rows; Cupertino glyphs → Lucide; the three totals keep the elevated SummaryCard treatment. The neutral-by-default discipline holds — every expense reads with the same muted glyph (no per-category colors invented), so color is reserved for status and only the destructive delete path carries red. The delete confirm adopts the shared dialog shell. Currency renders grouped (₱1,234.00), with full dark parity (gold leads).
