Handoff 11 · Suppliers · migration

Suppliers — on the new language

The supplier directory (list + add/edit form) migrated to the elevated system. Material Card + ListTile rows become soft-shadow AppCards; the scrolled Form is grouped into sectioned cards with a pinned submit; Cupertino icons → Lucide. Suppliers carry no status, so the surface stays strictly neutral — the only differentiation is the muted + strikethrough treatment for inactive rows. Full dark parity. No delete / deactivate UI is introduced — none exists on these screens.

AppCard rows — Material Card/ListTile → soft-shadow surface
Grouped form — sectioned cards + pinned submit
Lucide — replaces Cupertino
Neutral discipline — inactive = muted + strikethrough
Light theme
