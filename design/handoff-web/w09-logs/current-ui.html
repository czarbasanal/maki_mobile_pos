Read-only screen — no mutations. Rows show action (falls back to the type's display name), a right-aligned en-PH time, an optional details paragraph, and a meta line with a user icon + userName + a plain-grey role pill. The 9×9 type square is tinted by the activity's tone (see the full tone/icon tables in the README). Day headers stick to the top of the scroll region on a blurred background. Only admins reach /logs (gated by viewUserLogs) — staff/cashier never see the Activity Logs sidebar item and are bounced to Access Denied.
