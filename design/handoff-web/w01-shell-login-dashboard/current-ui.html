The AdminShell wraps every non-auth screen: a fixed 240px sidebar on the left, no top bar, and a scrolling content region on the right (with the OfflineBanner pinned above it when offline). The sidebar filters each item through canAccess — the same gate as the route guard — and hides any section whose items are all denied.
