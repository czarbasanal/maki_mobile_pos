# Inventory Transform Rules & Instructions

## Field Sequence
name, model, color, category, code, price, qty, unit, reorder_level, supplier

## Field Defaults
- Missing value → **NA**
- Missing qty → **1**
- reorder_level → **0**
- supplier → **NA**
- unit → specified per category group

---

## General Rules

### Bike Model Name Formatting
- No spaces in model/bike abbreviations
- Examples: SMASH110, CLICK125, XRM110, W100, MIO125, NMAX155, AEROX155, RAIDER150, TMX155, BEATFI, WAVE125

### Brand Aliases
- TGO TAKASAGO → **TGO**
- SHARK KING → **SHARKKING** (one word)
- GB → **GLOBAL**

### FR / RR Designation
- Use **FR** for front, **RR** for rear in item names

---

## Category Transform Rules

### CHAIN
- Name format: `CHAIN [BRAND] [SPEC]`
- Model: NA | Color: NA (unless explicitly stated)
- Category: **CHAIN & SPROCKET**
- Unit: **PC**
- Example: `CHAIN GLOBAL 428H-120L`

### ENGINE VALVE
- Name format: `ENGINE VALVE [BRAND] [SPEC/MODEL]`
- Model: NA | Color: NA
- Category: **ENGINE**
- Unit: **SET**
- Example: `ENGINE VALVE TGO SMASH110`

### CDI
- Name format: `CDI [BRAND] [MODEL]`
- Category: **POWER SUPPLY**
- Unit: **PC**

### PHONE HOLDER
- Name format: `PHONE HOLDER [BRAND]`
- Model: NA
- Category: **ACCESSORIES**
- Unit: **PC**

### HORN & SOCKET RELAY
- Name: `HORN RELAY W/ SOCKET [BRAND]`
- Category: **POWER SUPPLY**
- Unit: **SET**

### ADJUSTABLE FLASHER
- Name format: `ADJUSTABLE FLASHER [BRAND]`
- Category: **LIGHTS**
- Unit: **PC**

### HI TEMP GREASE
- Name format: `GREASE HI-TEMP [BRAND]`
- Generic: `GREASE ORD` (code: F, price: 20)
- Category: **LUBRICANTS**
- Unit: **PC**

### REPAIR KIT (Brake)
- Name format: `MASTER REP KIT FR/RR [BRAND] [MODEL]`
- Category: **BRAKE**
- Unit: **PC**
- SHARKKING = one word
- Merge duplicate brand+model+position entries (add qty)
- Example: `MASTER REP KIT FR SHARKKING R150`

### CARBURATOR REPAIR KIT
- Name format: `CARB REP KIT [BRAND] [MODEL]`
- Category: **FUEL SYSTEM**
- Unit: **SET**

### BUSHING
- Name format: `BUSHING [TYPE] [BRAND] [MODEL]`
- Types: SWING ARM, PIVOT, STEEL
- Category: **CHASSIS**
- Unit: **SET**
- Example: `BUSHING SWING ARM KHC SHOGUN/SMASH`

### PAINT
- Name format: `PAINT [BRAND] [MODIFIER if needed] [3-LETTER COLOR CODE]`
- Model: NA | Color: NA
- Category: **PAINT**
- Unit: **PC**
- Modifiers: HiTemp, FLAT, MEDIUM, FLUORESCENT (keep as-is)
- Special: "HONDA RED" kept as full name
- Example: `PAINT SAMURAI HiTemp BLK`, `PAINT SHERLUX MDM BLU`, `PAINT SHERLUX HONDA RED`

### SEAT ASSY
- Name format: `SEAT ASSY [MODEL]`
- Model: NA | Color: NA
- Category: **BODY PARTS**
- Unit: **PC**
- Example: `SEAT ASSY TMX155`

### OIL
- Name format: `[PRODUCT] [BRAND]`
- Model: NA | Color: NA
- Category: **LUBE**
- Unit: **PC**
- Includes: FORK OIL, BRAKE FLUID
- Example: `FORK OIL NATIONAL`, `BRAKE FLUID RS8`

### FLUIDS
- Name format: `[PRODUCT] [BRAND]`
- Model: NA | Color: NA
- Category: **FLUIDS**
- Unit: **PC**
- Includes: ANTI RUST, TIRE BLACK, SEALANT (from T-SEALANT), CARB CLEANER, CVT CLEANER
- Special: CVT CLEANING → `CVT CLEANER 450ML`
- Example: `SEALANT KOBY`, `CVT CLEANER 450ML KOBY`

### WASHER / LOCK WASHER
- Name format: `PLAIN WASHER [S/S or ORD] [SIZE]` / `LOCK WASHER [SIZE]`
- Model: NA | Color: NA
- Category: **BOLT&NUT**
- Unit: **PC**
- Example: `PLAIN WASHER ORD 12MM`, `PLAIN WASHER S/S 6MM`, `LOCK WASHER 8MM`

---

## Process Rules
- Go group by group iteratively
- Ask for unit and category if not yet defined
- Ask for clarification on brand or description if ambiguous
- Output as tab-separated text for direct paste into Google Sheets
- Record confirmed rules in memory before moving to next group
